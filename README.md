# Pidgey Escape

## Getting Started

1. Download or clone this project
2. `npm run install`
3. `npm run start`
4. `npm run webpack:watch`
5. Visit `localhost:8080`

## Deployment

1. `npm run webpack:build`
2. Output files are in `dist/`
